# 创建备份策略<a name="ZH-CN_TOPIC_0059304223"></a>

## 功能介绍<a name="section31829604"></a>

创建备份策略对服务器进行周期性备份，还可以为备份策略附加一个复制策略，对该策略产生的备份进行复制。

创建备份策略对服务器进行周期性备份。
